# Bank Server

## Error codes

- 0: No error
- 1: Invalid login (User not found or incorrect password)
- 2: Invalid registration (User already registered)
- 3: Insufficient funds (Tried to withdraw more than it has)

- 252: UUID not found
- 253: Bad arguments
- 254: Unknown command
- 255: Unknown error
