# telematics-project

## Error codes

- 0: No error
- 1: Unknown command
- 2: Invalid login (User not found or incorrect password)
- 3: Invalid registration (User already registered)
- 4: Not enough balance (Tried to withdraw more than it has)
- 5: Invalid transfer (Unknown account)
- 255: Unknown error
